## How to use with composer
create composer.json and write below

    {
        "require": {
            "mebius/mebius": "dev-master"
        },
        "repositories": [{
            "type": "vcs",
            "url": "https://github.com/mebius1080p/PHPLib.git"
        }]
    }

add comment to trigger travis
